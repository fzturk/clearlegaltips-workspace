# Local SEO Booster

## Context

ClearLegalTips has 5 state/specialty guides (IDs 176–180). However, "how to form an LLC in [state]" gets 500–8,000 monthly searches PER STATE. Expanding from 5 to 30+ state articles is one of the highest-ROI content moves available.

**Top 10 states by LLC formation search volume:**
Texas, California, Florida, New York, Georgia, North Carolina, Illinois, Ohio, Arizona, Colorado

**State-specific factors that differ:**
- LLC filing fee ($50–$500)
- Processing time (1 day – 6 weeks)
- Annual report requirement (yes/no, fee, due date)
- Registered agent requirements
- Publication requirement (NY, AZ, NE)
- Operating agreement requirement (CA, MO, NY, NE)
- State income tax on LLCs

## Instructions

### Step 1 — State Article Template Structure

Every state guide follows this structure:
```
H1: How to Form an LLC in [State]: Step-by-Step Guide ([Year])
H2: [State] LLC Formation at a Glance (summary table)
H2: Step 1 — Choose Your LLC Name
H2: Step 2 — Appoint a Registered Agent
H2: Step 3 — File Articles of Organization
  → State filing fee, form name, online/mail options
H2: Step 4 — Create an Operating Agreement
  → Required in [state]? / Recommended anyway
H2: Step 5 — Get Your EIN
H2: [State]-Specific Requirements
  → Publication requirement? Annual report? Franchise tax?
H2: How Much Does an LLC Cost in [State]?
  → Table: Filing fee + registered agent + annual report + total
H2: Best LLC Formation Services for [State]
  → Affiliate CTA section
H2: Frequently Asked Questions
H2: Final Thoughts
```

### Step 2 — State-Specific Data to Research

For each target state, gather:
```
State: [name]
Filing Agency: Secretary of State / [specific dept]
Filing Form Name: [Articles of Organization / Certificate of Formation]
Online Filing: YES/NO — URL
Mail Filing: YES/NO
Filing Fee: $[X]
Processing Time: [X] business days (standard) / [X] hours (expedited, $[fee])
Annual Report: YES/NO — due [date], fee $[X]
Registered Agent Requirement: Must have physical address in state
Publication Requirement: YES/NO — [details if yes]
Operating Agreement Required: YES/NO (legally)
State Income Tax on LLCs: YES/NO — [rate if yes]
```

### Step 3 — State Content Differentiation

Each state article must have at least ONE unique element:
- State-specific quirk (NY publication requirement costs $1,000–$2,000)
- State tax advantage (Wyoming: no income tax, no franchise tax)
- Processing speed note (Delaware: 1-hour expedited available)
- Specific form name (Texas: Certificate of Formation, not Articles of Organization)

### Step 4 — Affiliate Angle by State

| State | Best Affiliate Angle |
|---|---|
| States with publication requirement (NY, AZ) | "Use a service to handle the publication — it's complex" → Registered agent services |
| High-fee states (CA, TX) | "Compare total costs before choosing a formation service" |
| No-tax states (WY, NV, SD) | "Consider forming in Wyoming even if you live elsewhere" |
| All states | ZenBusiness or Northwest as default CTA |
